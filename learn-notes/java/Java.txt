1 继承（二）：
1）java中永远是单继承，一个子类只能有一个直接的父类，但是可以层层继承。
2）继承中的构造方法：
	a）子类可以继承父类的属性和方法，但不能继承付了的构造方法，可以调用父类的构造方法
	b）如果父类没有空参（没有参数）构造方法，则必须在子类的构造方法中，调用父类中带参数的构造方法，固定规则。
super：过去的，曾经的
super（参数）：在子类的构造方法中，调用父类中参数的构造方法，而且必须在子类的构造方法的第一行使用
	c）如果父类中含有参数的构造方法，则子类可以任意编写自己的构造方法，不需要强制的调用父类中的构造方法

	d）在企业开发中，一般都建议编写父类的时候最好能编写一个空参的构造方法，方便子类编写

2 变量的隐藏（属性覆盖）：子类定义的属性名跟父类中继承过来的属性名一致，子类定义的属性会覆盖父类值继承过来的属性

3 方法重写：子类编写的方法覆盖了从父类中继承过来的方法
1）方法重写：子类编写的方法名字和参数，跟父类中要重写的方法名字和参数一致。方法重写也称方法覆盖
2）参数一致：参数的类型和个数必须一致
